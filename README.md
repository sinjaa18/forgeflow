# ForgeFlow

ForgeFlow is a Sales Operations Suite for Manufacturing Teams built using the MERN stack.

## Features

- JWT Authentication
- Protected Routes
- Lead Management
- Sales Dashboard
- Pipeline Tracking
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- TailwindCSS

### Backend
- Node.js
- Express.js
- MongoDB
- JWT

## Run Locally

### Backend

```bash
cd server
npm install
npm run dev

### Frontend
cd client
npm install
npm run dev
Environment Variables

Create .env inside server folder:

MONGO_URI=your_mongodb_uri
JWT_SECRET=your_secret
